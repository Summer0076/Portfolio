Let's discovered

Description

Ce projet est une application Android permettant de réfléchir sur des questions à plusieurs pour se découvrir, ou bien même seul pour s'amuser.

Concept

Cette application mobile regroupe des questions de 3 types :
légère, intime ou profonde. Chaque type de question permet d'explorer une partie différente de l'application :
- intime pour les émotions,
- légère pour des questions simples,
- profonde pour de la réflexion.

Réalisation

Ce projet a été réalisé via React Native en TypeScript et Expo en 1 semaine.
